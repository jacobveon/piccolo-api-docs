JSON API

Command Wrapper

  • api_version - Use the latest api version available, e.g. 2.0.0.
  • ref_id - Custom string field that is echoed back in the response.
  • command - Command to execute.
  • payload - Payload json for the command.

Example Request

{
    "api_version": "2.0.0",
    "ref_id": "1",
    "command": "go",
    "payload": {
        "zone": "Zone1",
        "coord": "A1"
    }
}

Example Response

{
    "api_version": "2.0.0",
    "ref_id": "1",
    "success": true
}
